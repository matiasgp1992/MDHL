# MDHL — Mighty Ducks Hockey League

Website for the **Mighty Ducks Hockey League (MDHL)**, a youth hockey league serving children ages 4–12 in Chicago's northside neighborhoods. Built with HTML, CSS, and Responsive Design using Media Queries.

## Pages

| File | Description |
|------|-------------|
| `index.html` | Home page with upcoming events |
| `about.html` | Mission, Vision, and General Information |
| `game_info.html` | Fall Schedule with game dates, teams, locations, and maps |
| `rules.html` | Rules of play, sportsmanship, and IHF regulations |
| `registration.html` | Player registration form |
| `contact.html` | Contact information |
| `show_data.html` | Registration confirmation page |

## Tech Stack

- HTML5
- CSS3 (Flexbox, Media Queries)
- Vanilla JavaScript

## Responsive Design

The site uses a desktop-first approach with a breakpoint at 600px for mobile devices. Navigation includes a hamburger menu for small screens.
